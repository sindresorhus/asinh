# asinh

> ES2015 [`Math.asinh()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/asinh) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install --save asinh
```


## Usage

```js
var asinh = require('asinh');

asinh(1234);
//=> 7.811163549201245
```
